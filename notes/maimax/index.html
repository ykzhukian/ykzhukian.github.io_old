Maimax BACK

Maimax is a smart server to handle chatting, currency searching, weather inquiry, expenditure recording, voice to text translating, language translating and more. Pure back end this time. Node.js | Koa | LeanCloud | Baidu Translate | Juhe Data Api

Again, it is a self learning project for Node.js and I chose Koa as the framework to bootstrap. I have to say that it really separates all back end part from front end. It’s more effective for duty separation among a team.

Back to this project, it is also for my personal use via WeChat interface (sending text or voice message to the server via WeChat to do the works on server). I’ve learned generators along with a series of Promise and tried to use ES6 among the work to get familiar with the arrow functions in this project.
